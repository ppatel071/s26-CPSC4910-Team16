Admin Bulk Upload

Upload a pipe-delimited text file to create organizations, sponsor users, and driver users.




Format

<type>|organization name|first name|last name|email address|points|reason for points

  • Use O for organizations, D for drivers, and S for sponsor users.
  • Organizations must already exist or be created earlier in the same file with an O record before sponsor or driver records can use them.
  • If points are supplied for a driver, a reason is required.
  • Points on sponsor records are ignored and reported as warnings.
  • New users are created with username equal to email. Temporary passwords for newly created users appear below after the upload completes.

Example

O|New Organization

D|New Organization|Joe|Driver|me@email.com|100|New Driver

S|New Organization|Jill|Sponsor|jill@mail.com

D|Existing Organization|Tom|Smith|tom887@gmail.com
